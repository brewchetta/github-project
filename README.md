# Git Project

When building a readme you can check out this repo for inspirations: [matiassingers/awesome-readme](https://github.com/matiassingers/awesome-readme)

When building a readme you generally want to answer these questions:

1. What is the project about?
2. How do I use or install the project?
3. What technologies and libraries were used to make this project?
4. How do I contribute to the project if I want to help or notice a bug?

Additionally people might also include this information:
- What license are you using for your project? MIT is an example of a free license that's easy to use...
- Where do I find a version log so I know what's changed?
